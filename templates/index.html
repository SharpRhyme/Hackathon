GCP Connection Guide

Run the following commands in your terminal to initialize and connect this application with your Google Cloud account:

1
Configure Project ID 
./google-cloud-sdk/bin/gcloud config set project my-first-project
2
Authenticate User Account 
./google-cloud-sdk/bin/gcloud auth login
3
Generate Application Credentials

This allows local Python programs to authenticate automatically.

./google-cloud-sdk/bin/gcloud auth application-default login
